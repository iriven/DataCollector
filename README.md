# DataCollector
Iriven DataCollector is a PHP container for key/value pairs


> Usage:

$Author =  new \Iriven\ IrivenPHPDataCollector([
                            'name'          => $name,
                            'email'         => $email,
                            'homepage'      => $homepage,
                            'role'          => $role,
                            'description'   => $description
                        ]);
